# Anaconda basic shortcuts (minimul viable set)

## Short and Sweet
```
Shift + Enter -> Output
d, d -> delete cell
ctlr + shft + Up -> Up
ctlr + shft + Down -> Down
B -> insert cell below
A -> insert cell above

Markdown + # <Text> / ## <Text> and so on -> Heading -- uptill level 6
Bold **text** or __text__
italic *text* _text_
strike ~~text~~
inline code `text`
link [text](https://www.google.co.in)

shft + enter -> run
alt + enter -> run and insert below
ctrl + enter -> only run a cell

X -> Cut
C -> Copy
V -> Paster

shft + m -> merge selected cell
shift + l -> show line number
```
